About AI Agent Tester

          

            AI Agent Tester is a desktop application for Mac, designed to make it easier to run and test AI agent interactions with a more convenient UI than the command line.
          

          

            Please note: This application is a highly experimental side project likely to be broken. Check in with Wouter S.  for feedback and bugfixes.
          

        


        

          

            
          

          
Getting Started

          
    
            
  • Be aware that this is basically a NodeJS wrapper around various agent CLIs and terminal commands.
    • 
            
  • Both for initial agent config and debugging issues, the easiest way is likely to work in the Mac Terminal and then come back here.
    • 
            
  • Select a session from the sidebar to view conversations.
    • 
            
  • Use the command dropdown to choose agent and/or execute predefined commands.
    • 
            
  • Check the Settings page for themes, agent configuration options and more.
    • 
          

        


        

          

            
          

          
Example MCP Server

          

            This app includes a sample MCP (Model Context Protocol) server called "Famous Quotes" that provides access to famous quotes.
            The server is a standalone Node.js file (famous-quotes-mcp-server.js) with no dependencies.
            You can start/stop it and configure your AI agents to use it from the Settings page.
          

          

            Go to Settings → Example MCP Server to manage the server and get configuration instructions.
