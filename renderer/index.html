Help & About

About AI Agent Tester

AI Agent Tester is a desktop application for Mac, designed to make it easier to run and test AI agent interactions. This tool provides a convenient interface for executing commands, viewing agent responses, and managing test threads.

Experimental Status

Please note: This application is still in a highly experimental state. Features may change, and you might encounter bugs or unexpected behavior. Use with caution in production environments.

Developer & Feedback

Getting Started

  • Select a session from the sidebar to view conversations
  • Use the command dropdown to execute predefined commands
  • Type custom messages in the input area
  • Check the Settings page for configuration options (coming soon)
